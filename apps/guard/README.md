# apps/guard

PWA local-first para guardias de seguridad (usuario operativo primario de GateFlow). Offline-first por diseño — ver `CLAUDE.md` §12.

En Sprint 0 este directorio contiene solo el esqueleto inicial del proyecto (sin pantallas ni lógica de negocio). El manifest y service worker mínimos se añaden en la tarea S0-17. La base de datos local embebida y la integración con `packages/sync` quedan pendientes de sprints posteriores.
